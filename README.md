# DeskBreak

Программа-напоминалка для тех, кто много сидит за компьютером. Работает
в системном трее Windows и присылает уведомления: пора встать и
размяться, дать отдых глазам (правило 20-20-20), проверить осанку,
выпить воды. У каждого типа напоминаний свой настраиваемый интервал.
Есть встроенная библиотека упражнений с описаниями.

## Состав проекта

```
deskbreak/
├── main.py            — основной код приложения
├── exercises.py        — база упражнений и советов
├── requirements.txt     — зависимости Python
├── build.bat            — автоматическая сборка .exe и установщика
├── installer.iss        — скрипт Inno Setup (установщик)
└── assets/
    └── icon.ico          — иконка приложения
```

## Почему сборку нужно делать на Windows

Готовый код полностью рабочий, но собирать `.exe`-файл и установщик
нужно **на Windows** — это стандартная практика (кросс-компиляция
Windows-программ из Linux ненадёжна: нет доступа к нужным Windows API,
самому tkinter/trayicon для тестов, WinToast и т.д.). Весь процесс
занимает 5–10 минут.

## Пошаговая сборка (Windows)

1. Установите **Python 3.10+** с [python.org](https://www.python.org/downloads/)
   (при установке отметьте галочку "Add Python to PATH").
2. Установите **Inno Setup 6** — бесплатный инструмент для создания
   установщиков: https://jrsoftware.org/isdl.php
3. Скопируйте папку проекта на Windows-ПК.
4. Откройте папку проекта и запустите `build.bat` двойным кликом.
   Скрипт сам:
   - поставит нужные Python-библиотеки,
   - соберёт `dist\DeskBreak.exe` через PyInstaller,
   - соберёт установщик `installer_output\DeskBreak-Setup.exe` через Inno Setup.
5. Готово. `DeskBreak-Setup.exe` — это финальный установщик, который
   можно загружать на Softpedia.

Если предпочитаете вручную:

```bat
pip install -r requirements.txt
pyinstaller --noconfirm --onefile --windowed --icon=assets\icon.ico --add-data "assets;assets" --name DeskBreak main.py
iscc installer.iss
```

## Проверка перед публикацией

Перед отправкой на Softpedia обязательно:
1. Запустите `DeskBreak-Setup.exe` на чистой Windows-машине (или в
   виртуалке) и убедитесь, что установка, ярлыки и запуск работают.
2. Проверьте, что Windows Defender / SmartScreen не блокирует файл.
   Программа не подписана цифровой подписью (это платная услуга),
   поэтому при первом запуске Windows может показать предупреждение
   "Неизвестный издатель" — это нормально для несертифицированного ПО
   и не помешает публикации на Softpedia, но стоит упомянуть об этом
   в описании, если сайт спросит.
3. Сделайте 2–3 скриншота: иконка в трее, всплывающее уведомление,
   окно настроек, библиотека упражнений — они пригодятся для карточки
   на Softpedia.

## PAD-файл (Portable Application Description)

В архиве есть готовый **`PAD_File.xml`** — стандартный XML-файл с описанием
программы, который принимают большинство каталогов софта, включая Softpedia
(автоматически подтягивает из него название, описание, категорию, ключевые
слова и т.д., экономя ручной ввод в форме).

Перед отправкой открой `PAD_File.xml` в любом текстовом редакторе и замени
все места с пометкой `ЗАПОЛНИТЬ` / `ВАШЕ_...` / `ВАШ_...` на свои данные:

- `Company_Name` — твоё имя или название компании
- `Author_Email`, `Support_Email` и т.д. — контактный email
- `Company_WebSite_URL`, `Application_Info_URL` — сайт проекта (если нет —
  можно оставить пустым, но тогда часть каталогов может отклонить заявку)
- `Application_Screenshot_URL`, `Application_Icon_URL` — прямые ссылки на
  скриншот и иконку, размещённые в интернете (например, на своём сайте или
  в открытом репозитории — сам PAD-файл изображения не хранит, только ссылки)
- `Primary_Download_URL` — прямая ссылка на скачивание `DeskBreak-Setup.exe`
- `File_Size_Bytes`, `File_Size_K`, `File_Size_MB` — точный размер уже
  собранного `DeskBreak-Setup.exe` (посмотри в свойствах файла после сборки)
- `Filename_Versioned` — реальное имя файла установщика, если оно отличается

Остальные поля (описания на английском, ключевые слова, системные
требования, версия) уже заполнены и готовы к использованию.

- Иллюстрации/GIF для каждого упражнения вместо эмодзи.
